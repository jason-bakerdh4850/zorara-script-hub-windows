## Overview

Zorara is a Windows desktop tool for loading and executing Roblox Lua scripts through a straightforward application workflow. It is intended for users who need a quick way to run Luau scripts, explore an integrated script collection, and organize multiple script tasks from one interface.

The application is designed as a native Windows experience and includes automatic updates, a queue that remains available between sessions, and an integrated debugger. A script hub with more than 500 entries and support for multiple interface languages further extend its use on Windows 10 and 11.
